# Sominnum Malanova

Repositório principal do projeto do jogo **Sominnum Malanova**.

Um platformer 2D dark fantasy focado em exploração, sonhos, trauma psicológico e combate.

---

## Estrutura Principal

### Assets/
Contém todos os assets e sistemas utilizados no jogo.

Inclui:
- scripts
- animações
- sprites
- áudio
- prefabs
- cenas
- UI
- artbook

---

### Packages/
Pacotes e dependências utilizados pelo projeto Unity.

Gerenciado automaticamente pela Unity Package Manager.

---

### ProjectSettings/
Configurações gerais do projeto Unity.

Inclui:
- input system
- layers
- física
- renderização
- tags
- qualidade
- build settings

---

## Sobre o Projeto

O jogo acompanha um protagonista amaldiçoado por uma relíquia misteriosa capaz de transformar sonhos em realidades distorcidas.

Enquanto explora o mundo e enfrenta criaturas geradas pelos próprios traumas, o jogador descobre segredos sobre o reino, seus pais e a origem da relíquia conhecida como **Jester**.
