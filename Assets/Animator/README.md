# Animator

Esta pasta contém os Animator Controllers do projeto.

---

# Conteúdo

## PlayerAnimator.controller

Responsável pelas animações do jogador.

## EnemyAnimator.controller

Responsável pelas animações dos inimigos.

---

# Objetivo

Controlar:

* transições
* estados
* blend trees
* parâmetros de animação

utilizados pelos personagens e entidades do jogo.
